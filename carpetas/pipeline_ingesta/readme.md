### ¿Qué es un pipeline de datos?
 es un conjunto de componentes que permite obtener, transformar, almacenar y consultar datos para un determinado fin de forma secuencial. 

 ### Ejemplo de empresa que utiliza pipeline de datos
 Netfilx, por medio de un pipeline de datos obtiene información acerca del comportamiento de datos.

 ### ¿Qué pasa si se salta una etapa de datos?
 significaria en datos inexistentes o inutiles.
